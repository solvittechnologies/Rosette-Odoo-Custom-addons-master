Set salary rules as aggregate

In some contexts it is useful to filter out some payslip lines that are a sum of other rules.

E.g. if you want to present the payslip as an old-fashined sum, or as a pivot table with a grand total on the rightmost column, it makes sense to hide some summands so that the end result looks coherent.

On the other hand, these rules may still appear on places that don't sum up all the summands themselves (such as the default payslip print template) unless the 'Appears on the payslip' field is unset.

This is a very simple module published under the GNU LGPLv3, free of change.
