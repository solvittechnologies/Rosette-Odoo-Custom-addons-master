Material consumption Odoo App

This Application Is developed to record direct consumption of Material Used by the entity. Through this application you Can add material Consumed cost To Cost of Sales and deduct it from Inventory cost at specific warehouse or location.

Accounting Journal Entries related to cost of Sales and Inventory Deduction will automatically be created and you don't need to add manual adjustment and use shortcuts or temporary remedies for consumption now.

Demo Request

Material consumption.

It allows you to create material consumption request simply by clicking this menu item.

Create Material consumption Request.

Click the create button to create new Material consumption request.

Add Products on consumption Request.

User can select the location and Product they want to consume.

Add consumed Quantity.

User can add consumed quantity for each product and then click validate button to validate.

Accounting Entries of material consumption

A smart button will appear after validation and you can check the accounting entries generated.
