Compare and monitor payroll variation

Compare payroll variation between two periods and no more get confused when there is a variation

What is added by the module ? :


We add a menu "Report" in payroll module who contain two submenus:
  1. Compare payroll rule: A menu who launch a report who give you the difference between the payroll of the two period. The result will generate a document containing a table who give you all the selected rule and the variation found by rule.
  2. Compare payroll by employee: A menu who launch a report who give you the difference by employee between the payroll of the two period. Locate quickly in which payslip there is a difference this month. Instead of spendind a lot of time, get directly to the payslip which make the difference and exactly which rule change for him.



1) After insert the module in your addons folder go and search modules starting with "hyd_" and install the comparative payroll module




2) Here are some payslip created, two in june and one in may, we will find the difference between the two payroll using our reports very quickly




3) By the way the module have created a new menu report in payroll module let's click on the first sub menu "Comparative table payroll". CHoose the period 1 and the period 2 that you want to compare. You can filter the rule to be compare. When it is done click on print




4) Here are the result, we can see a variation in the BASIC rule (basic salary) about 250 000 more than the previous payroll, and in the net rule too. SO what happens ? using the next report we will find it easyly.




5) To have more details go back to menu report and select this time, the submenu "Compare payslip salary", fill the period fields, and use an filter this time select only the rule basic, gross and net. And print the report




5) Now we know that the employee davis simpson did not have his salary last month. That is the reason the payroll of may was different that the one in june.
